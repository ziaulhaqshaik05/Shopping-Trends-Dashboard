# Customer Shopping Trends Dashboard 🛒

## Project Overview
An interactive Excel dashboard analyzing customer shopping behavior 
using a dataset of 3900 real customer transactions.

## Key Features
- 4 KPI Cards showing Total Revenue, Total Customers, Average Purchase and Average Review Rating
- 4 Interactive Charts showing Category, Season, Gender and Payment Method analysis
- 4 Slicers for dynamic filtering by Season, Gender, Category and Age
- Clean professional design with consistent color theme

## Tools Used
- Microsoft Excel 2021
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas (SUM, AVERAGE, COUNTA)

## Dataset
- 3900 customer records
- 18 columns including Age, Gender, Category, Purchase Amount, Season, Payment Method

## Key Insights
- Clothing is the highest revenue category at $104,264
- Male customers spend more than double female customers
- Fall season generates the highest revenue at $60,018
- All payment methods are used equally by customers
